# Food_Ordering_Website
Food Ordering Website is primarily used for Ordering Food Online, tracking status of Orders.<br>

order food from their favorite Restaurant and It also allows the admin to keep track of orders, update orders, <br>
customers, maintain customer and review customer feedback. Developed this website for a particular restaurant <br>
(client) Requirements.
