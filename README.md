# Food_Ordering_Website
Food Ordering Website is primarily used for Ordering Food Online, tracking status of Orders. The scope of 
Food Ordering website involves user-friendly website that allows customers to browse menus, order food 
from their favorite Restaurant and It also allows the admin to keep track of orders, update orders, customers, 
maintain customer and review customer feedback. Developed this website for a particular restaurant (client) 
Requirements.
